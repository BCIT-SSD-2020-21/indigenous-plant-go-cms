## 2) CREATING A PLANT
- To create a new plant, click on Add New Plant from the dashboard or go to plants on the left menu and click Add New.
- Enter Plant name, Scientific name and Description
![](https://i.imgur.com/T9LMg2T.png)
- User can either add an existing location or create a new location. To add an existing location, choose between the drop down list and click Add Existing Location. To Create a New Location, click the + Create A New Location then enter Location Name, Latitude, Longitude and Description. Click Create new Location when finished. To remove the selected location, click the delete button on the right side of the location name.
![](https://i.imgur.com/5brKXJ1.png)
![](https://i.imgur.com/GGmXlVz.png)
- To add a Plant Image, user can either select an existing image or upload a new image. When adding an existing image, select between the drop down list and click Add Existing Image. To upload a new image click + Upload A New Image, enter Caption and choose the image file then click Upload Image. To remove the selected image, click the delete button on the right side of the image caption.
![](https://i.imgur.com/qoEmXOk.png)
- To add a Plant Audio File, user can either select an existing audio or upload a new audio file. When adding an existing audio file, select between the drop down list and click Add Existing Audio file. To upload a new audio file, click + Upload A New audio file, enter the Caption and choose the file then click Upload Audio File. To remove the selected audio file, click the delete button on the right side of the audio caption. 
![](https://i.imgur.com/POLMRwE.png)
- To add a Plant Video, user can either select an existing video or upload a new video. When adding an existing video, select between the drop down list and click Add Existing Video. To upload a new video click + Upload A New Video, enter the Caption and the Youtube link then click Upload Video. To remove the selected video, click the delete button on the right side of the video caption.
![](https://i.imgur.com/qyHgFdp.png)
- To add a Plant Category, user can either select an existing category or create a new category. When adding an existing category, select between the drop down list and click Add Existing category. To create a new category, click + Create A New Category, enter the category name and click Create New Category. To remove the selected category, click the delete button on the right side of the category name.
![](https://i.imgur.com/4cptTfq.png)
- To add a Plant Tag, user can either select an existing tag or create a new tag. When adding an existing tag, select between the drop down list and click Add Existing Tag. To create a new tag, click + Create A New Tag, enter Tag Name and click Create new Tag. To remove the selected tag, click the delete button on the right side of the tag name.
![](https://i.imgur.com/pf3ROi9.png)
- When the user is done adding changes, the last step is to click on Publish and a new plant will be successfully added. The user has an option to show or hide this resource on the mobile application by clicking the checkmark at the right bottom part of the system. 
![](https://i.imgur.com/a6MxCaI.png)

## 3) VIEWING A PLANT

- User can view the list of plants by clicking All Plants in the menu. The pagination on the bottom right can be clicked for the next list page of plants. 
![](https://i.imgur.com/mOoFFpa.png)
- User can perform bulk actions such as select or delete all. To select all, click the check box at the left side of the Title. 
![](https://i.imgur.com/GF9D44E.png)
- To edit or delete plants, click edit or delete. For editing, fill the details and click update. To delete the plant, click “Yes, i know what i am doing.” and the plant will be removed.
![](https://i.imgur.com/Q96c8CX.png)
![](https://i.imgur.com/onGD6Vj.png)
- To filter the plant based on category, select between the drop-down list and click filter.
![](https://i.imgur.com/8VD6cNt.png)
- To search for a particular plant, enter the plant name in the search box and click search.
- ![](https://i.imgur.com/LEVxCoc.png)
- View, Create, Edit or Delete plant category via categories in menu. To add a category, fill the category name and click Create new category. 
![](https://i.imgur.com/tlK6Gji.png)
- To edit or delete, click between the two button below the plant name. User can select plants individually or delete all by clicking the check box beside the name.
![](https://i.imgur.com/vUWMvme.png)
![](https://i.imgur.com/zfZIYad.png)
![](https://i.imgur.com/F4fCpHC.png)
- User can search categories in search box using category name.
![](https://i.imgur.com/K8EwuaS.png)

## 4) VIEWING A WAYPOINT
- To view waypoints, go to All waypoints.
![](https://i.imgur.com/MGPQhnC.png)
- To filter the waypoints based on category, select between the drop-down list and click filter. 
![](https://i.imgur.com/7X8OhfQ.png)
- To search for a particular waypoint, enter the waypoint name in the search box and click search. 
![](https://i.imgur.com/ZUcH9GZ.png)
- To add a new waypoint, fill the waypoint details and click on publish to save changes. 
![](https://i.imgur.com/qNqPpbT.png)
- To edit or update waypoint details click edit, fill additional changes, and click on update.
![](https://i.imgur.com/QtUWvvJ.png)
- To view waypoint categories, click on Categories. User can create a new waypoint category by entering category name and click Create new category. 
![](https://i.imgur.com/bclQzUP.png)
- To select and delete all waypoint categories, select delete, click on the checkbox and click apply.
![](https://i.imgur.com/KyEzrGS.png)
- To search for a waypoint category enter the category name on the search box and click on search. 
![](https://i.imgur.com/J7F37CW.png)

## 5) VIEWING A USER
- To view the list of users, click on users on the menu and go to All Users. Click on the pagination on the bottom right to go to the next list of pages.
![](https://i.imgur.com/6dPLfUr.png)

![](https://i.imgur.com/v9e0Akl.png)
- To delete a user, click on the delete button under the username and click “Yes, I know what i am doing.” 
![](https://i.imgur.com/hpWWXdj.png)
- To delete all user, select delete on the drop down list, click on the checkbox beside the username and all the users will be highlighted. Click on apply to delete all.
![](https://i.imgur.com/tJ4DOYX.png)
- To search for a user, enter the user name and click on search. User can also filter the other users according to their roles by selecting roles on the drop down list and click filter. 
![](https://i.imgur.com/CBe28bu.png)
- To create a new user, go to Add New and fill up the form for details and  click on the Register User button and the new user will be created.
